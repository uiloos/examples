This example shows a classic carousel, it is classic because it does not use modern CSS (with snap points) to create the carousel.

Instead it uses JavaScript to create the carousel, which is why it does not work when JavaScript is disabled.

The user can either use the chevron buttons to navigate between the slides, or the user can click on the progress indicators. Once the user takes control of the carousel it will stop auto playing.

The idea behind this example is to show that "uiloos" is aware of the position an item has within the ActiveList. "uiloos" can tell you what the next item, previous item and what first and last items are.

You can then use this information to set different CSS classes for each item.

This example shows that via "maxActivationLimit" you can limit the amount of content / items that can be active.

By setting "maxActivationLimitBehavior" to "ignore" whenever more items are activated than the "maxActivationLimit" they are simply ignored.
